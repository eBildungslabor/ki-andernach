Erkundung 5: Mit einem KI-Sprachmodell Perspektivwechsel üben

Normalerweise würden Sie sich von einem KI-Sprachmodell einen Inhalt zur Weiternutzung generieren lassen. Alternativ können Sie mit einem KI-Sprachmodell auch Spiele spielen und sich so in einen offenen Prozess des Lernens begeben. Hier ist zum Beispiel ein anpassbarer Prompt für ein Spiel, mit dem Sie sich in einem Perspektivwechsel üben können. Lesen Sie ihn sich gut durch, damit Sie Ihre Rolle korrekt spielen - sonst funktioniert der Prompt nicht. Insbesondere müssen Sie mit einer ersten fiktiven Rolle und einem Zitat starten.

Ich möchte mit dir ein Spiel spielen, das folgendermaßen funktioniert.
1. Wir sind immer abwechselnd an der Reihe. Ich starte, dann du, dann schreibe ich wieder etwas, dann du ... Das Spiel endet, wenn ich schreibe: Das Spiel ist zu Ende!
2. Wir nehmen in unseren Beiträgen immer eine zufällig ausgewählte Rolle aus der Erwachsenenbildung ein. Das kann eine Dozentin, ein Teilnehmer, eine Leiterin eines Bildungswerks .. sein. Sie können unterschiedliche Ausprägungen haben (z.B. eine überforderte Dozentin, eine Teilnehmerin, die Digitalisierung skeptisch gegenüber steht ...) In jedem Beitrag entscheiden wir uns für eine andere Rolle.
3. Wir bauen unseren Beitrag folgendermaßen auf: Ich bin [fiktiver Name und Infos zu meinen Charakteristika]. Ich habe von dir diese Aussage gehört: [Wiederholung der Aussage von dem Beitrag davor]. Ich denke dazu: [Ganz kurze Erläuterung der Perspektive]. Das bringt mich stattdessen zu dieser Aussage: [neue Aussage]
4. Die Beiträge sollen in sich logisch formuliert sein und immer eine veränderte Perspektive zum vorherigen Beitrag aufmachen. Der erste Beitrag wird von mir stammen. Da es der erste Beitrag ist, kann ich da noch nicht auf einen früheren Beitrag Bezug nehmen.
Hast du das Spiel verstanden und kann ich mit dem ersten Beitrag starten?

Frei und ohne Registrierung zugängliche KI-Sprachmodelle finden Sie z.B. über diesen Link bei der Suchmaschine DuckDuckGo.

Dieses Spiel ist exemplarisch zu verstehen. Wenn Ihnen das Prinzip klar ist, können Sie beliebig eigene 'Spiele' erfinden. Ausführlichere Informationen finden Sie dazu in diesem Blogbeitrag.
